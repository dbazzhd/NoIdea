Open the folder LUA_Unity_Level_Exporter_UNITY_C# in Unity.
The root object contains a script that will export the name, position and color of its child objects to a lua script file.
The file is called "exported_scene.lua" and will be saved in the project folder.

You assignment is to run this script file in MGE. The download is available on Blackboard. The lastest version will be published on the Blackboard page of the 3D Rendering cluster.
